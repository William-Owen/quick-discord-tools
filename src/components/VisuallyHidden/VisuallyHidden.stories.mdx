# VisuallyHidden

```javascript
import { VisuallyHidden } from "pvh-core-ui"
```

## Overview

This utility component is designed to hide content from the
scree, whilst allowing it to be accessible to assistive
technologies such as screen readers. This can be useful for
providing access to context that might otherwise be
conveyed visually in a manner inaccessible to assistive
technologies. For example size and colour.

## Props

<Props of={VisuallyHidden} />

## Styling Guide

Do not style this component directly. If the component is
set to show on focus then the children of the component can be
styled.

## Usage Guide

None

### Show on focus VisuallyHidden

When set the content of the component will be shown when it
receives focus. This can be useful to provide additional
information to people who are using assistive technologies
to navigate the page. For example, providing links to
quickly jump to content for those navigating via a keyboard.

nothing shown in the below example. It is recommended that
it is viewed in dev tools.

		showOnFocus: true
	}}>
		{TemplateWithChildren.bind({})}
	</Story>
</Canvas>
